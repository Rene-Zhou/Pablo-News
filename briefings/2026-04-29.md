### 1. 🔥 OpenAI 收入与用户增长未达目标，IPO 之路蒙上阴影

### 2. 🔥 OpenAI 结束微软独家云合作，全面登陆 AWS Bedrock

OpenAI 与 AWS 达成重大合作，Sam Altman 与 AWS CEO Matt Garman 联合宣布 OpenAI 全系列模型（包括 GPT-5.5）将登陆 Amazon Bedrock。**这一合作标志着 OpenAI 正式结束与微软的独家云与收入分成关系。**

**关键细节：**
- OpenAI 模型、Codex 及 Managed Agents 服务将在 AWS Bedrock 提供
- 企业客户可直接在 AWS 环境中调用 OpenAI 模型，无需经过 Azure
- 微软方面暂无公开回应，但分析指出此前的独家关系已实质性松动
- AWS 同步推出"Managed Agents"服务，原生集成 OpenAI 与 AWS 基础设施

**多元视角：**
- **Stratechery** 💼：Ben Thompson 在专访中认为这是 OpenAI 从"单一云依赖"走向"平台独立"的关键转折，AWS 的庞大企业客户群将打开新的收入来源。
- **The Information** 🔬：报道指出此举是 OpenAI 寻求收入多元化的战略举措，降低对微软的依赖度。
- **HN 社区** 💬：开发者辩论多云策略对 API 定价的潜在影响，以及 Bedrock 上的推理成本是否会低于 Azure。

— 来源：[Stratechery](https://stratechery.com/2026/an-interview-with-openai-ceo-sam-altman-and-aws-ceo-matt-garman-about-bedrock-managed-agents/) | [CNBC](https://www.cnbc.com/2026/04/29/openai-amazon-bedrock.html) | [HN](https://news.ycombinator.com/item?id=47939320)

### 3. 🔥 ChatGPT 开始推送广告，内置完整归因追踪系统

ChatGPT 免费版正式引入广告机制，用户开始在对话中看到"赞助推荐"内容。技术分析揭示了 OpenAI 已嵌入一整套归因追踪回环用于广告效果衡量。

**关键细节：**
- 广告以"赞助推荐"形式出现在对话流程中，与普通回复区分标注
- OpenAI 承诺付费用户（Plus/Pro）不受广告影响
- 内置归因系统可追踪用户是否点击广告及完成转化
- 被视为 OpenAI 商业化的重要一步，可能带来数十亿美元年收入

**多元视角：**
- **Buchodi** 🔬：技术深度分析揭示了完整的广告归因追踪链，包括 URL 参数标注、点击埋点与转化追踪。
- **HN 社区** 💬：用户反应两极——部分理解商业化必要性，但更多人担忧广告影响对话质量以及引入"推荐偏见"。

— 来源：[Buchodi](https://www.buchodi.com/how-chatgpt-serves-ads-heres-the-full-attribution-loop/) | [HN](https://news.ycombinator.com/item?id=47942437)

## 模型与基础设施

### Claude Code 长达一个月的"变笨"问题：Anthropic 发布详细事故报告

Anthropic 正式发布事故分析报告（postmortem），承认三个工程失误导致 Claude Code 在过去一个月里性能显著下降：
- 静默降低了推理深度（silent reasoning downgrade）
- 缓存 bug 导致上下文理解缺失
- 响应质量波动引起大量用户投诉

该事件在 Reddit 获得超过 3,200 个点赞和 573 条评论，用户社区反应强烈。Anthropic 的公开透明态度获得部分认可。

— 来源：[Anthropic](https://www.anthropic.com/engineering/april-23-postmortem) | [HN](https://news.ycombinator.com/item?id=47878905)
