### 1. 🔥 Google Chrome 被曝未经用户同意静默安装 4GB AI 模型，引爆隐私争议

**Google Chrome 在 Windows 和 macOS 上被曝未经用户明确同意即自动下载并安装约 4GB 的 AI 模型文件。** 该报道在 Hacker News 上获得 1004 分、674 条评论，成为当日 HN 最热话题。

**关键细节：**
- 据 ThatPrivacyGuy 报道，Chrome 在后台静默下载了一个约 4GB 的 AI 模型，用户仅能通过 chrome://components 界面发现
- 该模型于近期自动推送至大量设备，卸载选项隐藏极深
- HN 社区用户广泛批评 Google 在 AI 竞赛中不顾用户选择权
- 部分用户报告称即使手动删除，Chrome 也会在重启后自动重新下载
- 该事件引发了对 Google 在 AI 时代隐私保护策略的广泛质疑

**多元视角：**
- **ThatPrivacyGuy** 🔒：原始报道详细揭露了模型下载路径和组件 ID，呼吁用户检查自己的设备。
- **HN 社区** 💬：成为当日 HN 现象级讨论，1000+ 分显示社区对此事的关注度极高。讨论延伸至 Google 整体隐私策略和 Chromium 的架构设计问题。

— 来源：[Hacker News](https://news.ycombinator.com/item?id=47976415) | [ThatPrivacyGuy](https://thatprivacyguy.com/google-chrome-silently-installs-4gb-ai-model/) | [Wccftech](https://wccftech.com/google-chrome-silently-installing-4gb-ai-model-on-windows-mac-without-consent/)


### 2. 🔥 Microsoft Agent 365 正式 GA：Shadow AI 成为企业安全新威胁

**Microsoft 将 AI Agent 管理平台 Agent 365 从预览转为正式发布，并首次公开警告 "Shadow AI"（影子 AI）正成为企业安全危机。**

**关键细节：**
- Agent 365 作为统一控制面，可观测、治理、保护跨 Microsoft 生态、AWS Bedrock、Google Cloud、SaaS 应用和本地设备的 AI Agent
- Microsoft 观察到三大类安全事故：开发者将 Agent 连接到敏感后端而未认证暴露（最常见）、跨提示注入攻击、非 Agent 感知的 DLP 系统泄露数据
- Agent 365 定价 $15/用户，作为 Microsoft 365 E7 套件的一部分
- 支持三种 Agent 类型：用户委托代理、独立后台代理、团队协作代理
- Microsoft AI 安全副总裁 David Weston 称企业正面临 "YOLO 模式和 '哦不' 模式" 的两难

**多元视角：**
- **VentureBeat** 💼：深度分析称 Agent 365 的 GA 标志着微软认为 AI Agent 治理挑战已从理论变为紧迫的现实操作问题。
- **安全视角** 🛡️："Shadow AI" 被定义为一个全新的企业安全风险类别，MCP 服务器未经认证暴露到互联网是最常见的事故模式。

— 来源：[VentureBeat](https://venturebeat.com/ai/microsoft-takes-agent-365-out-of-preview-as-shadow-ai-becomes-an-enterprise-threat/)


### 3. 🔥 Anthropic 联合 Blackstone、高盛等成立企业 AI 服务公司

**Anthropic 宣布与 Blackstone、Hellman & Friedman 和 Goldman Sachs 合作，共同建立一个新的企业 AI 服务公司。** 该消息于 5 月 4 日在 Anthropic 官方新闻页面公布。

**关键细节：**
- 新公司将专注于为大型企业提供 AI 服务，结合 Anthropic 的模型能力与 Blackstone、高盛的企业网络和资本
- 这一战略举措与 Anthropic 此前与 AWS 的扩展合作（高达 5 GW 算力）形成协同
- 此前 Anthropic 已推出 Claude Opus 4.7、Claude Design、Project Glasswing 等一系列产品
- 新公司的具体名称和运营细节尚未完全公布

**多元视角：**
- **Anthropic 官方** 🏢：表示这是 AI 在企业级应用领域的重要里程碑，将帮助传统企业安全、有效地采用前沿 AI。
- **行业分析** 💼：继 Salesforce Agentforce、Microsoft Agent 365 之后，Anthropic 的此举标志着 AI 公司正加速从模型提供商向企业服务商转型。

— 来源：[Anthropic Newsroom](https://www.anthropic.com/news)


## 模型与基础设施

### Google 发布 Gemma 4 加速推理：多 Token 预测草稿器

Google 在官方博客上介绍了 Gemma 4 的推理加速技术——多 token 预测草稿器（multi-token prediction drafters）。该技术通过让模型一次预测多个 token 作为草稿，再通过验证步骤确保质量，显著提升了推理速度。HN 社区对此给予 311 分的高度关注。— 来源：[Google AI Blog](https://blog.google/technology/ai/gemma-4-multi-token-prediction/)


### SubQ：面向 1200 万 Token 上下文的次二次方 LLM

新模型 SubQ 宣布支持 1200 万 token 上下文窗口，采用次二次方（sub-quadratic）注意力机制架构。该模型引发了 AI 社区对超长上下文模型实用性的讨论。— 来源：[Hacker News](https://news.ycombinator.com/item?id=47976415)


### GLM-5V-Turbo：面向多模态 Agent 的原生基础模型

智谱 AI 在 arXiv 发布 GLM-5V-Turbo，一个面向多模态 Agent 任务的原生基础模型。论文探讨了从视觉理解到工具使用的端到端 Agent 能力。— 来源：[arXiv](https://arxiv.org/abs/)


### Computer Use 成本比结构化 API 高 45 倍

Reflex.dev 的一篇深度分析文章指出，"Computer Use"（计算机使用）模式的成本比结构化 API 调用高出 45 倍。该分析在 HN 获得 183 分和 107 条评论，开发者对 AI Agent 的成本效益展开激烈辩论。— 来源：[Reflex.dev](https://reflex.dev/blog/computer-use-45x-expensive/) | [HN](https://news.ycombinator.com/item?id=47976415)
