### 1. 🔥 Microsoft 连发 MAI-Code-1-Flash 和 MAI-Thinking-1 两款新模型

**Microsoft 于 6 月 2 日发布两款全新 AI 模型——MAI-Code-1-Flash（198 分，90 条评论）和 MAI-Thinking-1（67 分，16 条评论），前者专注于代码生成，后者主打深度推理能力，标志着 Microsoft 在自研 AI 模型方向迈出重要一步。**

**关键细节：**
- MAI-Code-1-Flash 定位为面向开发者的编码模型，与 GitHub Copilot 生态深度整合，旨在与 OpenAI Codex 和 Anthropic Claude 在编程场景中竞争
- MAI-Thinking-1 基于论文《main》发布，采用新的推理架构，在数学和逻辑推理任务上展现竞争力
- 两款模型同日发布标志着 Microsoft AI 正在从「OpenAI 的独家合作伙伴」向「拥有自主模型矩阵的平台方」转型——MAI 系列已成为覆盖编码、推理和 Agent 能力的完整模型家族
- 此前 Microsoft 已发布多款 MAI 系列模型（包括面向 Agent 的 Scout 等），此次发布进一步加速了模型发布的节奏
- HN 社区对 Microsoft 自研模型的关注度显著上升，部分开发者认为 MAI 系列的实际表现「值得认真对待」

### 2. 🔥 Adafruit 收到 Flux.ai 律师函：因揭露服务器配置漏洞被威胁起诉，暂停博客更新

**Adafruit 于 5 月 22 日收到来自 Fenwick & West 律师事务所（由前 FBI 幕僚长 Jonathan Lenzner 代理）的律师函，要求其不得发表一篇关于 Flux.ai 服务器配置漏洞的调查报告。** Adafruit 公布了这封律师函的全文，称已暂时停止博客更新，正在评估应对方案。该消息在 HN 上获得 531 分和 225 条评论，成为当日最受关注的科技话题。

**关键细节：**
- Flux.ai 的律师函指责 Adafruit 计划发表的报道包含「虚假且具有诽谤性」的指控，涉及 Flux 的知识产权、商业规模和用户基础
- 律师函还援引《计算机欺诈与滥用法案》（CFAA）主张权利
- Adafruit 坚决驳斥这些指控，称其仅访问了 Flux 自身系统因配置错误而公开暴露的信息，调查「涉及公共安全利益」，属于负责任的披露流程
- 事件引发了对「安全研究人员何时会被律师函威胁」的广泛讨论——Adafruit 是知名的开源硬件和创客社区平台，很少面临此类法律威胁
- 225 条 HN 评论中，大量安全社区成员表达了担忧：如果连 Adafruit 这样声誉卓著的组织都会被以 CFAA 威胁，独立安全研究的环境将进一步恶化

— 来源：[Adafruit Blog](https://blog.adafruit.com/) | [HN](https://news.ycombinator.com/item?id=48368121)

### 3. 🔥 Anthropic 扩展 Project Glasswing 至 15+ 国：AI 漏洞扫描覆盖关键基础设施

**Anthropic 宣布将 Project Glasswing 扩展至约 150 家新组织，覆盖 15 个以上国家。** 该项目利用 Claude Mythos 模型对关键软件基础设施进行大规模漏洞扫描，已发现超过 10,000 个高危或严重安全漏洞。该消息在 HN 上获得 130 分和 173 条评论。

**关键细节：**
- Project Glasswing 于今年 4 月初启动，最初仅约 50 家合作伙伴参与，如今规模扩大了 3 倍
- 合作伙伴来自多个国家，涵盖开源软件维护者、安全行业和美国政府机构
- Claude Mythos Preview 模型专门针对代码安全审计进行了优化，已在多家组织的代码库中发现超过 10,000 个高危/严重漏洞
- Anthropic 表示正在与安全行业和开源社区密切合作，以确保漏洞得到及时修复
- TechCrunch 报道（44 分）指出，Claude Mythos 已扩展到 15 个国家的关键基础设施领域，包括能源、金融、通信等行业
- 173 条评论中，安全社区对 AI 辅助漏洞发现的效果和效率展开了讨论

— 来源：[Anthropic](https://www.anthropic.com/news/expanding-project-glasswing) | [TechCrunch](https://techcrunch.com/2026/06/02/anthropic-scales-claude-mythos-to-critical-infrastructure-in-15-countries/) | [HN](https://news.ycombinator.com/item?id=48369863)

### 4. Microsoft 发布 Scout 自主 AI Agent：内部文件称要「让人上瘾」

**Microsoft 正式发布 Scout，一款基于 OpenClaw 框架构建的自主 AI Agent，同时 404 Media 曝光了其内部规划文件——文件明确将「让人上瘾」列为产品目标。** 该消息在 Computer World 报道（37 分，29 条评论）和 404 Media 报道（22 分，3 条评论）中从不同角度呈现。

**关键细节：**

- 404 Media 获得的内部规划文档显示，Microsoft 的策略是「先让用户对 Scout 上瘾，再逐步添加新功能」——这一表述引发了隐私和成瘾性设计的担忧

- 此前 Microsoft 已通过 Copilot 在办公场景广泛部署 AI，Scout 则进一步瞄准自主任务执行场景
- HN 社区对「设计上瘾」的评论持批评态度，认为这与科技行业此前对「成瘾性设计」的反思形成矛盾
