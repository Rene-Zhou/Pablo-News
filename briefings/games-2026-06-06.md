### 2. 🔥 《生化危机：代号维罗妮卡》重制版正式公布，卡普空经典回归

**在夏日游戏节发布会上，卡普空正式揭晓了《生化危机：代号维罗妮卡》重制版（Resident Evil: Code Veronica Remake），结束了长达数月的传闻期。** 该作将是《生化危机》系列又一款经典作品的重制，延续《生化危机 2/3/4》重制版的路线。预告片展示了 Claire Redfield 在洛克福德岛上的经典恐怖冒险场景，采用 RE Engine 全面重制，画面表现力大幅提升。游戏定档 2027 年，将面向全平台发售。

关键细节：
- 定档 2027 年，登陆全平台
- RE Engine 全面重制
- 延续《生化 2/3/4》重制版路线，并非简单 HD 移植
- Claire Redfield 回归为主角

— 来源：[游民星空](https://www.gamersky.com/news/202606/2152294.shtml)


### 3. 🔥 Valve 宣布 Steam Machine 今夏发售，同步推出 Steam Frame VR 头显

**Valve 正式表态，此前被推迟的 Steam Machine PC 主机以及 Steam Frame VR 头显将在今夏正式发售。** 据 The Verge 报道，Valve 确认两款设备均已做好上市准备，并将引入"Verified"认证体系来标识游戏的兼容性表现，类似于 Steam Deck 的验证模式。Steam Machine 作为 Valve 重新进入客厅 PC 游戏市场的战略产品，距初代 Steam Machine 计划已过去十年，受到了 Hacker News 社区的广泛关注和讨论（67 分，13 条评论）。

关键细节：
- Steam Machine PC 与 Steam Frame VR 头显今夏发售
- 引入"Verified"认证标识游戏兼容性
- 此前因技术原因多次推迟
- HN 社区讨论热烈，67 分 13 条评论

— 来源：[The Verge](https://www.theverge.com/games/943657/valve-steam-machine-frame-summer-launch-verified) | [HN](https://news.ycombinator.com/item?id=48413420)

## 行业与商业

### 11. Paramount Games Studio 成立，整合 Skydance 游戏业务

**Paramount Skydance 正式宣布成立 Paramount Games Studio，将旗下 Skydance Interactive 与 Skydance New Media 并入统一游戏部门。** 据 Gematsu 报道，新工作室将在统一的品牌下开发 AAA 级游戏，整合双方在互动娱乐与叙事媒体方面的资源。Skydance New Media 由《神秘海域》系列编剧 Amy Hennig 领导，正在开发漫威《1943：崛起九头蛇》等作品；Skydance Interactive 则是《行尸走肉：圣徒与 sinners》系列的开发商。这一整合是 Paramount 在娱乐 IP 游戏化战略上的重要一步。

关键细节：
- 统一 Skydance Interactive + Skydance New Media
- Amy Hennig 领导团队正在开发漫威《1943：崛起九头蛇》
- Paramount 加速游戏 IP 战略布局

— 来源：[Gematsu](https://www.gematsu.com/2026/06/paramount-games-studio-formed-unites-skydance-interactive-and-skydance-new-media)


### 12. 国内最大 Steam 黑产窝点被捣毁，12 人卖出 1.8 亿单

**据国内媒体报道，一个涉及 Steam 平台黑产的大型犯罪团伙被捣毁，12 人累计非法交易 1.8 亿单。** 该团伙通过批量注册账号、利用漏洞低价购入游戏、跨区代购等方式进行非法牟利，严重破坏了 Steam 平台正常的市场秩序。此次执法行动标志着国内对游戏数字资产黑产的打击力度进一步升级。

— 来源：[游民星空](https://www.gamersky.com/news/202606/2150508.shtml)
