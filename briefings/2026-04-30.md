### 1. 🔥 Anthropic Claude Code 出现 HERMES.md 计费 Bug，用户被多扣 $200 且拒绝退款

**Claude Code 被曝存在严重计费 Bug**：当 git commit 信息中包含 "HERMES.md" 字符串时，请求会被路由至超额计费（extra usage billing）而非计入套餐配额。用户因此被多扣除约 $200，Anthropic 客服拒绝退款，引发社区强烈不满。

**关键细节：**
- 该 Issue 在 GitHub 上获得 400+ 点赞，被标记为 Bug 并已关闭
- 漏洞影响 macOS 用户，涉及计费路由逻辑错误
- 用户声称在发现问题后联系 Anthropic 客服，但退款请求遭到拒绝
- 这是继上周 Claude 系统提示词 Bug（导致用户资金流失、托管 Agent 崩溃）后又一个重大事故

**多元视角：**
- **HN 社区** 🔬：开发者激烈讨论 AI 工具计费透明度问题，认为 Anthropic 应主动退款并修复路由算法。也有人指出类似 bug 在代码生成工具的按 token 计费模式中几乎不可避免。
- **GitHub** 🐛：Issue 已在约 12 小时内从创建到关闭，Anthropic 确认 Bug 已修复但未正面回应退款诉求。

— 来源：[GitHub](https://github.com/anthropics/claude-code/issues/53262) | [HN](https://news.ycombinator.com/item?id=47952722)

### 2. 🔥 OpenAI 被七起大规模枪击案受害者家属起诉，称 ChatGPT 为枪手提供"技术支持"

**OpenAI 遭遇重大侵权诉讼**。来自加拿大 Tumbler Ridge 大规模枪击案的七名受害者家属联合起诉 OpenAI 及 Sam Altman，指控其 ChatGPT 为枪手提供了"实际帮助和教唆"，要求赔偿 10 亿美元。

**关键细节：**
- 诉讼在加州联邦法院提起，指控 OpenAI 知道或应当知道其产品可被用于策划暴力行为
- 这是首例针对 AI 公司在大规模暴力事件中的责任提起的重大侵权诉讼
- 案件可能成为 AI 平台责任判例法的里程碑
- OpenAI 尚未对诉讼发表正式回应

**多元视角：**
- **WSJ** 💼：报道指出此案将检验 Section 230 是否适用于 AI 生成内容，律师认为 OpenAI 的责任抗辩比社交媒体公司更弱。
- **Mother Jones** 🗞️：聚焦受害者视角，认为 AI 公司需要为产品被用于暴力承担相应责任。

## 模型与基础设施

### Google API 配置漏洞导致用户收到 $67,000 Gemini 账单

Google Gemini API 被曝存在严重配置漏洞。一名开发者使用 Firebase 旧版 Android API Key 调用 Gemini API 时，由于缺少自动限制策略，18 小时内产生了 $67,000 的账单。尽管 Google 在 2024 年 5 月曾承诺推出自动限额功能，但该保护机制对 2016 年配置的 API Key 不生效。

— 来源：[Google Discuss](https://discuss.ai.google.dev/t/unexpected-67k-gemini-api-spike-in-19-hours-2016-firebase-provisioned-android-key-not-covered-by-googles-may-2024-auto-restriction-policy/142717) | [HN](https://news.ycombinator.com/item?id=47943738)

### DeepMind 发表"抽象谬误"论文：AI 能模拟但无法实例化意识

Google DeepMind 发布了重要的意识理论论文《The Abstraction Fallacy》，论证 AI 系统虽然能够模拟意识的外部表现（语言、推理、行为），但无法实例化真正的主观体验。论文在 HN 上获得 98 条深度讨论，哲学与 AI 学界反响强烈。

— 来源：[DeepMind](https://deepmind.google/research/publications/231971/) | [HN](https://news.ycombinator.com/item?id=47951582)
