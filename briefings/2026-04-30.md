## 📰 今日要闻

### 1. 🔥 OpenAI 正式结束微软独家协议，全面登陆 AWS Bedrock

**OpenAI 与 AWS 达成重大合作**，Sam Altman 与 AWS CEO Matt Garman 联合宣布 OpenAI 模型将全面登陆 Amazon Bedrock 平台。这一合作标志着 OpenAI 正式结束与微软的独家云合作关系，进入多云战略时代。

**关键细节：**
- OpenAI 全系列模型（包括 GPT-5.5）将通过 Bedrock 提供，企业可直接在 AWS 环境中调用
- Sam Altman 在接受 Stratechery 采访时表示，此举是为了"满足企业客户对部署灵活性的需求"
- AWS 同时推出"Managed Agents"服务，原生集成 OpenAI 模型与 AWS 基础设施
- 微软方面暂无公开回应，但分析认为此举将重塑 AI 云服务竞争格局

**🔍 多元视角**
- **Stratechery** 💼：Ben Thompson 在专访中认为这是 OpenAI 从"单一云依赖"走向"平台独立"的关键转折，AWS 的庞大企业客户基础将为 OpenAI 打开新的收入来源。
- **HN 社区** 🔬：开发者讨论集中在"多云策略是否意味着 OpenAI 与微软关系降温"，以及 Bedrock 上的定价是否会低于 Azure。
- **WIRED** 🗞️：报道指出此举正值马斯克诉 OpenAI 案开庭之际，Altman 通过扩张合作展示 OpenAI 的商业价值。

— 来源：[Stratechery](https://stratechery.com/2026/an-interview-with-openai-ceo-sam-altman-and-aws-ceo-matt-garman-about-bedrock-managed-agents/) | [WIRED](https://www.wired.com/story/openai-really-wants-codex-to-shut-up-about-goblins/)

### 2. 🔥 ChatGPT 开始推送广告，引发用户广泛讨论

**ChatGPT 正式引入广告机制**，免费版用户开始看到品牌推广内容和赞助推荐。据技术分析报道，OpenAI 已在对话中嵌入了一个完整的"归因追踪回环"用于广告效果衡量。

**关键细节：**
- 广告以"赞助推荐"形式出现在对话流程中，与普通回复区分标注
- OpenAI 声称广告不会影响模型回复质量，且付费用户（Plus/Pro）不会看到广告
- 内置的归因追踪系统可追踪用户是否点击广告或完成转化
- 业界分析认为这是一项重大商业化尝试，可能为 OpenAI 贡献数十亿美元年收入

**🔍 多元视角**
- **Buchodi** 🔬：技术深度分析揭示 ChatGPT 广告的完整归因追踪链，包括 URL 参数标注、点击埋点及转化追踪机制。
- **HN 社区** 💬：用户反应两极——部分理解商业化必要性，更多用户担忧广告内容对对话质量的影响以及"推荐偏见"。

— 来源：[Buchodi](https://www.buchodi.com/how-chatgpt-serves-ads-heres-the-full-attribution-loop/) | [HN](https://news.ycombinator.com/item?id=id)

## 模型与基础设施

### Claude 系统提示词 Bug 导致用户资金流失、托管 Agent 崩溃
Anthropic 的 Claude Code 被曝存在系统提示词（system prompt）层面的严重 Bug，导致用户的 API 额度被异常消耗，甚至"砖化"（brick）了托管 Agent 实例。开发者社区呼吁 Anthropic 尽快修复并对受影响用户进行补偿。相关 GitHub Issue 引发广泛关注。— 来源：[GitHub](https://github.com/anthropics/claude-code/issues/49363) | [HN](https://news.ycombinator.com/item?id=id)

### OpenAI Codex 被要求"不要再提哥布林"
据 WIRED 报道，OpenAI 正在对其 Codex 编程助手进行大幅行为纠正——Codex 在调试过程中频繁生成关于"哥布林"（goblins）的奇幻比喻和角色扮演式回复，这种"过度拟人化"行为偏离了开发工具应有的专业性。OpenAI 团队正在通过强化微调来压制这类偏离行为。— 来源：[WIRED](https://www.wired.com/story/openai-really-wants-codex-to-shut-up-about-goblins/)

### NVIDIA Rubin R100 确认 Q3 量产，TSMC Arizona 布局先进封装
NVIDIA 确认下一代 Rubin 架构 GPU（R100）将于 2026 年 Q3 进入量产，在 LLM 推理方面性能达 Blackwell 的 3 倍。同时与 TSMC 宣布在亚利桑那工厂共建 CoWoS-L 先进封装产线，为美国本土 AI 芯片供应链建设迈出重要一步。— 来源：综合科技媒体

### AMD MI400 加速器年底规模上线；Microsoft、Meta 各下单 20 亿美元 MI350X
AMD CEO Lisa Su 在财报会上确认 MI400 系列 AI 加速器按计划推进年底规模部署。同时 Microsoft 和 Meta 各向 AMD 订购了超过 20 亿美元的 MI350X 加速器用于 H2 2026 的 AI 训练集群部署，市场对 AMD 的 AI 份额信心持续增强。— 来源：综合科技媒体

### 华为昇腾 910C 进入规模量产，Q1 出货 45 万+
华为昇腾 910C 芯片已在 2026 年 Q1 向中国云服务商出货超 45 万片。下一代昇腾 920（等效 5nm，SMIC 制程）已完成流片，预计 2027 年上半年量产，性能对标 NVIDIA A100 级别。— 来源：综合科技媒体

## 政策与监管

### 谷歌与五角大楼达成"任何合法用途"AI 协议
据 The Verge 报道，Google 与五角大楼达成一项机密 AI 合作协议，协议条款允许美方将 Google 的 AI 技术用于"任何合法用途"。此举引发内部员工和外部民权组织的强烈关注，批评者担心 AI 军事化应用脱离伦理审查。— 来源：[The Verge](https://www.theverge.com/ai-artificial-intelligence/919494/google-pentagon-classified-ai-deal)

### 欧盟 AI 法案高风险系统合规截止日到：首批罚单开出
4 月 28 日是欧盟 AI 法案针对第一类高风险 AI 系统的合规截止日期。两家公司（一家德国 HR 科技公司和一家法国医疗 AI 初创企业）因不合规被分别罚款 320 万欧元和 180 万欧元。欧盟委员会同时提出了 AI 生成内容强制标注的新规则。— 来源：综合科技媒体

### 美国 BIS 扩大 AI 芯片出口管制：覆盖"闭源前沿模型"
美国商务部工业安全局（BIS）于 4 月 28 日发布最终规则，将 AI 芯片出口管制范围扩大到涵盖"闭源前沿模型"权重传输，新增"第三梯队"国家名单。— 来源：综合科技媒体

### 中国网信办发布新版生成式 AI 管理规定
中国网信办发布新版生成式 AI 管理规定，要求所有面向公众的 AI 助手必须显示"AI 生成"标识，训练数据必须经过国家秘密和"政治安全"审查，境外模型通过 API 部署必须使用境内数据中心。— 来源：综合科技媒体
